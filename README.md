# Pokemon Go Status

A simple gem that will tell you if Pokemon Go is available.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pokemon-go-status'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pokemon-go-status

## Usage

    $ pokemon-go-status
